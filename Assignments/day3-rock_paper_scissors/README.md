## 🎮 Rock Paper Scissors App

> 간단한 가위바위보 게임 웹 애플리케이션입니다. <br><br>
> 사용자는 가위 / 바위 / 보 중 하나를 선택하여 컴퓨터와 대결하며 총 10번의 기회 동안 승점을 겨루게 됩니다.
> 게임이 종료되면 승패 결과를 확인한 뒤 다시 시작 버튼으로 게임을 초기화할 수 있습니다.

## ⁘ 기능 소개
#### 1. 플레이어와 컴퓨터의 선택 비교
플레이어가 고른 가위·바위·보와 컴퓨터의 랜덤 선택을 비교해 즉시 승패를 판정합니다.  
라운드 결과는 바로 화면에 표시됩니다.

#### 2. 제한된 라운드 진행
게임은 총 10라운드로 진행되며 매 라운드마다 남은 횟수가 줄어듭니다.  
모든 기회를 소진하면 자동으로 게임이 종료됩니다.

#### 3. 점수 업데이트
각 라운드 결과에 따라 플레이어와 컴퓨터의 점수가 실시간으로 반영됩니다.

#### 4. 최종 결과 판정
10라운드가 끝나면 두 점수를 비교해 최종 승리, 패배, 무승부 여부를 알려줍니다.

#### 5. 다시 시작 기능
게임이 종료되면 선택 버튼이 사라지고 ‘다시 시작’ 버튼이 나타납니다.  
버튼을 누르면 페이지가 새로고침되어 게임이 초기화됩니다.
