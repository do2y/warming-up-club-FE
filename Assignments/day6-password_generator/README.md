## 🔐 Password Generator App

> 원하는 규칙을 선택해 임의의 비밀번호를 생성하고 생성된 비밀번호를 클립보드에 바로 복사할 수 있는 비밀번호 생성 웹 애플리케이션입니다. <br>
> 숫자, 소문자, 대문자, 특수문자 포함 여부를 선택해 다양한 강도의 비밀번호를 만들 수 있습니다.

## ⁘ 기능 소개

#### 1. 비밀번호 길이 설정  
사용자가 입력한 길이(5~70자)를 기반으로 비밀번호가 생성됩니다.  
권장 길이를 안내하여 더 강력한 비밀번호 선택을 도와줍니다.

#### 2. 문자 종류 선택  
아래 문자 조합을 선택해 비밀번호 구성 규칙을 지정할 수 있습니다.  
- Numbers (0–9)  
- Small Letters (a–z)  
- Capital Letters (A–Z)  
- Symbols (@, !, #, $, &, %)  

#### 3. 비밀번호 생성  
선택된 문자 범위를 모두 합쳐 문자열 풀을 구성한 뒤, 랜덤 인덱스를 활용해 지정된 길이만큼 비밀번호를 생성합니다.  
결과는 화면 상단 영역에 표시됩니다.

#### 4. 비밀번호 복사 기능  
‘복사’ 버튼 클릭 시  
`navigator.clipboard.writeText()` API를 통해 비밀번호가 즉시 복사됩니다.  
비밀번호가 없을 경우 안내 메시지를 출력합니다.

#### 5. 입력 검증  
- 문자 타입이 하나도 선택되지 않은 경우 경고(alert)를 출력해 잘못된 요청을 방지  
- 비밀번호 길이가 비어 있거나 숫자가 아닐 경우 생성되지 않도록 처리  

<br>

## ⁘ 기술적 구현 특징
- 체크된 옵션을 기반으로 문자 세트를 동적으로 구성해 다양한 규칙의 비밀번호를 생성할 수 있도록 설계했습니다.  
- 랜덤 문자열 생성 로직을 `getRandomPassword()` 함수로 분리해 재사용성과 유지보수성을 높였습니다.  
- `navigator.clipboard` API를 활용해 비밀번호 복사가 클릭 한 번으로 이루어지도록 구성했습니다.
