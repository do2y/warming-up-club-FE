## 🧮 Math Quiz App

> 랜덤 산술 문제를 풀며 정답 여부를 확인할 수 있는 퀴즈 웹 애플리케이션입니다. <br>
> 문제는 덧셈, 뺄셈, 곱셈 중 하나로 출제되며 3번 이상 틀리거나 5문제를 모두 풀면 게임이 종료됩니다.

## ⁘ 기능 소개

#### 1. 랜덤 문제 생성  
덧셈(+), 뺄셈(-), 곱셈(x) 중 하나의 연산자와 두 개의 난수를 조합해 문제를 생성합니다.  
정답과 오답을 섞어 3개의 보기를 랜덤 배치합니다.

#### 2. 정답 판별  
사용자가 선택한 보기와 실제 정답을 비교하여  
- 정답이면 초록색 표시  
- 오답이면 빨간색 표시, 오답 카운트 증가  

즉각적인 피드백을 제공합니다.

#### 3. 문제 진행 및 종료 조건  
문제를 풀 때마다 다음 문제 버튼이 활성화되며,  
다음 조건 중 하나라도 충족하면 퀴즈가 종료됩니다:  
- 오답 3회 이상  
- 총 5문제 출제 완료  

종료 시 ‘restart’ 버튼이 나타나며 새로 시작할 수 있습니다.

#### 4. 사용자 인터랙션 제어  
정답을 선택하면 모든 선택지가 비활성화되어  
중복 선택을 방지합니다.

#### 5. 퀴즈 재시작  
restart 버튼 클릭 시 페이지를 새로고침하여 초기 상태로 돌아갑니다.
