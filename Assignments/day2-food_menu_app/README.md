## 🍽️  Food Menu App
> 간단한 음식 메뉴를 카테고리별로 필터링하여 보여주는 웹 애플리케이션입니다. <br>
> 사용자는 상단의 버튼을 클릭해 ALL / Breakfast / Lunch / Dinner / Shakes 중 원하는 카테고리를 선택하면 해당되는 음식만 화면에 표시됩니다.
